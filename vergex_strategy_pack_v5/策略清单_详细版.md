# 策略清单（详细版，优化版）

下面所有策略都按 VergeX Studio 的 **第 2-6 步** 写出，方便直接照着填。
推荐顺序：**S1 → S2 / S3 → S4 / S5 → S6 → S7**

---

## 先看这条 UI 规则
- **Custom**：可以用 `Select Tokens` 做正向白名单
- **非 Custom**：不支持正向白名单
- **Excluded Coins**：永远是排除币种（黑名单）
- 因此，AI500 / Netflow / OI / Gainers / Losers 这类策略里的“优先做哪些币”，统一写进 **Prompt**

Source Type：Custom

说明：只有 Custom 支持正向白名单；这套严格只做 BTC/ETH。

### 3. Basic Rules
- AI Decision Interval：60 min
- Position Mode：Isolated Margin
- Max BTC/ETH Leverage：3x
- Max Altcoin Leverage：2x
- Max Account Leverage：1x
- Hard Stop：8%
- K-Line Data：15m, 1h, 4h
- Technical Indicators：RSI14, MACD, BB, EMA20, EMA50, ATR
- Stats：OI, Netflow

### 4. Prompt Logic 核心思路
- 交易优先级：BTCUSDT / ETHUSDT
- Excluded Coins 建议：不适用；这套直接用 Select Tokens 做正向白名单
- 逻辑摘要：4H 定方向，1H 看结构，15m 只做回撤入场；多数周期允许 WAIT
- 风险边界：不追单边大阳/大阴；RR 至少 1:3；信号不清就不做
