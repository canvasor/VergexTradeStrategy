# 版本优化说明

本版是针对原始 `vergex_strategy_pack.zip` 的优化版，主要吸收了后续 v2 / v3 的两类经验。

## 一、结构上保留了什么
- 保留原始 **S1-S7** 的多 Source Type 分层结构
- 保留“从保守到激进”的测试顺序
- 保留新手优先从 S1 起步的主结论

## 二、修正了什么
### 1. 非 Custom 不支持正向白名单
原始包里一些非 Custom 策略写了“建议优先主流币”的思路，这次统一拆成两层：
- UI 层：`Excluded Coins` 只做黑名单
- Prompt 层：写“优先交易哪些币”

### 2. 杠杆字段改成当前 UI 更兼容的整数
- 不再使用 1.5x / 1.2x 这类小数 Max Account Leverage
- 统一改成 1x 或 2x，并通过更严格的 Hard Stop 和 Prompt 来补风险控制

### 3. Prompt 模板整体升级
所有策略都增加了：
- `Trade only symbols that are currently in the selected source universe`
- `Do not analyze, select, or trade any symbol listed in Excluded Coins`
- `If there is no valid setup, output WAIT`
- `Prefer fewer high-quality trades`
- `Respect the account-level hard stop`

### 4. 更贴合实盘行为
参考后续实际运行观察，这版更强调：
- 允许大量 WAIT
- 必须给出 trigger level / invalidation
- 高波动 source 下不追离均值太远的价格
- 接近账户 hard stop 时优先保本，不强行找机会

## 三、怎么理解这版
这版不是把原始包“推翻重做”，而是把它升级成：
- 更符合当前 VergeX UI 行为
- 更不容易因为白名单/黑名单搞错
- 更适合直接拿去小仓跑一段时间
