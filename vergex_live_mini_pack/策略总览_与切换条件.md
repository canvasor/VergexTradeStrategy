# 策略总览与切换条件

## 0. 适用前提
- 目标是小资金实盘，不追求最高收益截图
- 允许大量 `WAIT`
- 策略核心是：**先活下来，再扩大样本**

## 1. 三套主策略

### A1｜AI500 双周期趋势回撤 PRO
- **定位**：默认主跑
- **Source Type**：AI500 Data Provider
- **Excluded Coins**：默认留空，只在复盘后加入持续假突破、异常点差、异常插针币
- **Decision Interval**：60 min
- **Margin**：Isolated
- **Leverage**：BTC/ETH 4x，Alt 2x，Account 2x
- **Hard Stop**：12%
- **K-Line**：15m / 1h / 4h
- **指标**：RSI14 / MACD / BB / EMA20 / EMA50 / ATR / OI / Netflow
- **核心**：4H 定方向，1H 认结构，15m 只做回撤与确认
- **优点**：最稳，最符合当前实际运行结果
- **缺点**：容易大量 WAIT，行情爆发初期可能偏慢

### A2｜AI500 社区 Trend 改良版
- **定位**：当 AI500 整体趋势很清晰时接替或补充 A1
- **Source Type**：AI500 Data Provider
- **Excluded Coins**：默认留空
- **Decision Interval**：4h
- **Margin**：Isolated
- **Leverage**：BTC/ETH 5x，Alt 3x，Account 2x
- **Hard Stop**：15%
- **K-Line**：15m / 1h / 4h
- **指标**：RSI14 / MACD / EMA20 / EMA50 / ATR / OI / Netflow
- **核心**：更贴近社区 trend strategy，强调中频趋势跟随
- **优点**：更少噪音、更少决策频率
- **缺点**：更依赖大趋势，不适合横盘震荡

### S3｜Netflow Top 资金流顺势
- **定位**：当资金流领导性很强时作为补充
- **Source Type**：Netflow Top
- **Excluded Coins**：默认留空，只在复盘后排除 price / OI / netflow 经常冲突的币
- **Decision Interval**：30 min
- **Margin**：Isolated
- **Leverage**：BTC/ETH 4x，Alt 2x，Account 2x
- **Hard Stop**：10%
- **K-Line**：15m / 1h / 4h
- **指标**：RSI14 / MACD / EMA20 / EMA50 / ATR / BB / OI / Netflow
- **核心**：只做价格、OI、净流入方向一致的标的
- **优点**：更适合抓轮动和资金主导的顺势机会
- **缺点**：当资金流信号反复打架时会变钝

## 2. 推荐排序
1. **A1**：稳定度 86 / 100
2. **S3**：稳定度 79 / 100
3. **A2**：稳定度 75 / 100

## 3. 切换条件
### 继续跑 A1
- AI500 候选池里主流币和次主流币结构都偏混乱
- 账户回撤已经明显，需要更保守
- 你更希望“少错”而不是“多做”

### 从 A1 切到 A2
- AI500 整体趋势持续清晰
- 4H 级别多数币同向
- 你接受更慢但更偏趋势的策略节奏

### 从 A1 / A2 切到 S3
- 价格与资金流 / OI 的领导性特别明显
- AI500 池虽然有机会，但噪音币较多
- 你想让资金流成为更强的过滤器

### 暂停所有策略
- 账户收益接近 Hard Stop，距离不足 4 个百分点
- 最近 2–3 笔都是明显低质量交易
- BTC / ETH 同时进入高噪音、无方向状态
- 平台 thought 经常偏离 Prompt 的核心原则

## 4. Excluded Coins 维护规则
只把这些币加入 Excluded Coins：
- 连续出现假突破 / 假跌破
- 点差显著更差
- 插针异常大，常把合理止损洗掉
- 经常 price / OI / netflow 三者互相冲突

不要把这些币加入 Excluded Coins：
- 只是暂时没信号
- 只是你“更想优先做”
- 只是今天没走出来

## 5. 实盘复盘清单
每次复盘至少看 5 个问题：
1. 这一轮是因为结构不清而 WAIT，还是模型偷懒而 WAIT？
2. 如果开仓了，是否真的满足 4H / 1H / 15m 多周期条件？
3. 是否出现追单边扩展、而不是等回撤？
4. thought 里有没有把 Excluded Coins 当成白名单误用？
5. 账户接近 Hard Stop 时，策略是否明显变保守？

## 6. 建议
- 默认只跑一套主策略
- 任何新改动只改一个变量
- Prompt、杠杆、Decision Interval 不要同轮一起改
