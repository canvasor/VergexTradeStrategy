# VergeX 精简实盘包（A1 / A2 / S3）

这个包只保留当前最值得实盘观察的 3 套策略：

- **A1｜AI500 双周期趋势回撤 PRO**：主跑，默认策略
- **A2｜AI500 社区 Trend 改良版**：中频趋势跟随，适合 AI500 整体方向很清晰时
- **S3｜Netflow Top 资金流顺势**：当资金流主导明显、AI500 候选池较噪时作为补充

## 先记住 4 条 UI/产品规则
1. **非 Custom source 不支持正向白名单**
2. **Excluded Coins 永远是黑名单**
3. **Max Account Leverage 用整数更兼容**，建议直接填 `2x`
4. **Run Test 参考价值有限**，优先看真实运行轮次里的 thought / action 是否符合 Prompt

## 默认启动顺序
1. 先只开 **A1**
2. 连续观察 8–16 个决策周期
3. A1 如果大量 WAIT 且 AI500 趋势扩散明显，再考虑开 **A2**
4. 如果 AI500 池噪音大，但资金流和 OI 共振很明显，再切到 **S3**

## 一键结论
- 想稳一点：**A1**
- 想更贴近社区 trend：**A2**
- 想抓资金流主导的顺势轮动：**S3**
