# 策略总览与使用说明

## 一句话结论

这版策略池收敛为 7 套：2 套 AI500 做多、2 套方向性补充做多/做空、1 套社区 F2 优化做空、1 套自适应多空伪网格。

## 为什么这样收敛

- **AI500 做多只留两套**：避免同类型策略过多而重复。
- **F2 替代旧 S2 TopLosers 反弹失败做空**：更容易标准化成“先冲、后弱、再回落”的结构。
- **Netflow Top / Low 更适合做方向过滤，不适合原生静态网格**。
- **自适应网格改用 Custom 主流币**：更符合主流币流动性和区间特征。

## 默认策略顺序

| 顺位 | 策略 | 适合场景 |
|---|---|---|
| 1 | A1 AI500 顺势持有 | 主线趋势明确、希望少交易 |
| 2 | L1 Netflow Top 资金流顺势 | 强势轮动、做最强的 1 个 |
| 3 | S1 Netflow Low / OI Decrease 弱势延续 | 空头市场或局部弱势币连跌 |
| 4 | A2 AI500 冲刺确认 | 想提高开单率，但仍需确认 |
| 5 | S2 Top Gainers 衰竭做空 | 涨幅榜币种冲顶衰竭 |
| 6 | L2 Top Gainers 强势延续 | 龙头币高位整理后的续涨 |
| 7 | G1 自适应多空伪网格 | 主流币震荡区间 |

## 策略切换建议

- A1 长时间不开单，但 AI500 仍有主线：切到 A2
- L1 开始频繁遇到冲高回落：切到 L2 或 S2
- S1 出现连续挤空反弹：暂停空头，观察 A1 / A2
- 市场进入明显震荡：切到 G1
- 市场强趋势恢复：从 G1 退出，回到 A1 / L1 / S1

## 网格策略说明

这版把原先的 G1 / G2 拆分网格合并成 **G1 自适应多空伪网格**。它不是交易所原生挂单网格，而是：

- 区间明确时，靠近边缘做低吸高减或高空低减
- 一旦区间失效或转单边，立即停止网格思路

更适合 Custom 主流币，不建议直接套在涨跌幅榜或极端资金流榜单上。
