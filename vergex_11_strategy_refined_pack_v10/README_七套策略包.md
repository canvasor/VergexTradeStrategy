# VergeX 中文收敛策略包（7套）

这是基于你前面多轮优化、运行问题复盘，以及最新社区开源策略 F2 整理的收敛版策略包。

## 这版的收敛原则
1. **AI500 做多只保留两套**：一套偏保守持有，一套偏激进冲刺。
2. **做空只保留两套主逻辑**：弱势延续做空、涨幅榜衰竭做空。
3. **保留两套高实用补充策略**：Netflow Top 顺势做多、Top Gainers 强势延续做多。
4. **网格只保留一个自适应伪网格**：改用 `Custom` 主流币，避免极端资金流榜单和低流动性噪音。
5. **统一修复历史问题**：
   - 非 Custom source 不再写白名单
   - `Excluded Coins` 只当黑名单
   - Prompt 统一中文、短而硬
   - 不写死固定资金与名义持仓
   - 显式写出 `WAIT` 逻辑，降低长期不开单和乱开单两种极端
   - 风险、杠杆、最小下单限制一律交给系统约束 + Prompt 二次强调

## 默认主跑顺序
1. **A1｜AI500 顺势持有**
2. **L1｜Netflow Top 资金流顺势**
3. **S1｜Netflow Low / OI Decrease 弱势延续做空**
4. **A2｜AI500 冲刺确认**
5. **S2｜Top Gainers 衰竭做空**
6. **L2｜Top Gainers 强势延续**
7. **G1｜自适应多空伪网格**

## 哪套最值得先用
- 想先稳：`A1`
- 想做资金流顺势：`L1`
- 想做更标准的空头：`S1`
- 想提升开单率：`A2`
- 想抓情绪见顶后的回落：`S2`
- 想做区间轮动：`G1`

## 文件说明
- `VergeX_AI500_七套收敛策略包.docx`：主手册
- `策略总览_与使用说明.md`：总览与切换建议
- `逐项填写版_Studio配置清单.md`：可直接照着 VergeX Studio 填写
- `策略清单_详细版.md`：每套策略的详细说明
- `策略排序_与配置矩阵.csv`：参数总表
- `版本优化说明.md`：本次收敛版相对前版的优化要点
- `prompts/`：所有中文 Prompt 文件

## 使用提醒
- 新策略先小资金跑
- 先看 24–72 小时运行特征，再决定是否加仓或切换
- 非 Custom 类型不支持正向白名单，优先把问题币放进 `Excluded Coins`
