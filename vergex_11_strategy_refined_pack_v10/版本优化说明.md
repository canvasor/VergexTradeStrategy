# 版本优化说明

## 本次相对上一版的主要变化
1. **收敛到 7 套**
   - 删除重复度高、定位过近的策略
   - 只保留 2 套 AI500 做多主策略

2. **F2 正式加入并优化**
   - 新增 `S2｜Top Gainers 衰竭做空｜优化版`
   - 不再使用旧版 `TopLosers 反弹失败做空` 作为主推

3. **网格策略重构**
   - 不再分别保留 `偏多伪网格 / 偏空伪网格`
   - 合并为 `G1｜自适应多空伪网格`
   - 改用 `Custom` 主流币，提升流动性与区间适配度

4. **全部 Prompt 继续沿用中文极简风格**
   - 不重复系统已自动注入的数据
   - 只写角色、方向、入场、退出、WAIT、风险边界

5. **统一修复历史问题**
   - 非 Custom 无白名单
   - `Excluded Coins` = 黑名单
   - 不写死固定资金
   - 避免过度复杂导致长期不开单
   - 避免规则过松导致乱开单
